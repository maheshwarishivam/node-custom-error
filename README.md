# node-custom-error
Custom Errors for NodeJS

## Installation
````
npm install node-custom-error
````
## Usage
````
var MyCustomError = require('node-custom-error');
function doSomethingBad() {
  throw new MyCustomError('It went bad!', 42);
}
````
## Tests
````
npm test
````
